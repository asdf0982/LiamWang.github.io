Posted on 2017-05-18 | Post modified 2017-05-18 | hits

环境说明

  • springmvc-4.3.6 jar包
  • jackson-2.7.4 [jar]包

代码说明

在springmvc.xml文件中采用<mvc:annotation-driven/>注解驱动。
在jsp文件下,配置一个按钮调用下面的js代码

在后台创建一个controller。传入json数据，@RequestBody转为实体类。传出实体类，使用@ResponseBody转为json格式。

在浏览器中打开jsp页面，F12控制台观察requestJson.action的状态码

问题

1.状态码200/415，不进入后端controller断点，浏览器弹出网页html代码
解决办法：缺少class，更换jar包。

2.状态码400/500，不进入后端controller断点，浏览器无反应
解决办法：jsp页面中data数据需采用正确的json编写格式，以逗号分隔。
